## What this project does

This repo contains the following Aliucord plugin modules under [plugins](plugins):

| Plugin | Version | Purpose |
| --- | --- | --- |
| `A2DPSucks` | `1.0.0` | Patches Discord audio behavior to improve Bluetooth/A2DP routing and voice communication handling. |
| `BoxUpload` | `1.3.2` | Uploads large attachments to external hosts (Catbox/Litterbox/Pomf) and inserts resulting URLs into chat. |
| `FixNameMention` | `1.0.0` | Fixes mention autocomplete collisions by preventing false equality between distinct user autocomplete objects. |
| `SelectableText` | `0.2.0` | Makes selected Discord text surfaces selectable (embeds, profile fields, rich presence, channel topic). |
| `StreamPip` (`SelfPip` class) | `1.0.0` | Experimental floating stream window (overlay-style PiP behavior while app is backgrounded). |

Project configuration is centralized in [plugins/build.gradle.kts](plugins/build.gradle.kts), with versions managed in [gradle/libs.versions.toml](gradle/libs.versions.toml).

## Getting started

### Requirements

- JDK 21.
- Android SDK (for building).
- Git (obviously).

### Clone

```bash
git clone https://github.com/FoxeiZ/aliucord-plugins.git
cd aliucord-plugins
```

### Build all plugins

Linux/macOS:

```bash
./gradlew make generateUpdaterJson
```

Windows PowerShell:

```powershell
.\gradlew.bat make generateUpdaterJson
```

### Build a single plugin

Linux/macOS:

```bash
./gradlew :plugins:BoxUpload:make
```

Windows PowerShell:

```powershell
.\gradlew.bat :plugins:BoxUpload:make
```

### Deploy to a connected Android device

Linux/macOS:

```bash
./gradlew :plugins:BoxUpload:deployWithAdb
```

Windows PowerShell:

```powershell
.\gradlew.bat :plugins:BoxUpload:deployWithAdb
```

### Build outputs

- Plugin zip files: `plugins/<PluginName>/build/outputs/*.zip`
- Updater metadata: `build/outputs/updater.json` (or module output depending on task execution)

If you consume this repo via Aliucord updater infrastructure, use the `builds` branch artifacts generated by CI.
